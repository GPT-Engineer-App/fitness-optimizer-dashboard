# fitness-optimizer-dashboard

Build an app that is essentially a dashboard that tracks how optimized your fitness journey is towards achieving your goal of losing fat or gaining muscle. The dashboard measures how effective the following 5 components are towards achieving your goal: diet, workout frequency, recovery time, sleep, and supplements

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/fitness-optimizer-dashboard.git
cd fitness-optimizer-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
